2. Description of Service

OpenDroid is an open-source, autonomous AI assistant for Android devices. The App can:

  • Execute system-level actions (toggle settings, set alarms, manage calendar events)
  • Make phone calls and send text messages on your behalf
  • Automate third-party applications via Android Accessibility Services
  • Capture and analyze screen content for task execution
  • Store memories and preferences locally on your device
  • Connect to third-party LLM API services using your own API keys
